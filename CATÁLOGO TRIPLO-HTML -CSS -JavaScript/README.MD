# Catálogo Completo de HTML e CSS

Este projeto foi criado como **material de estudo** para o curso de HTML e CSS.  
Ele contém exemplos práticos de praticamente todas as **tags HTML5** e das principais **propriedades CSS modernas**, organizadas em arquivos de referência.

---

## 📂 Estrutura do Projeto
- `index.html` → Arquivo de teste com todas as tags HTML5 comentadas.
- `style.css` → Arquivo de estilos demonstrativos aplicando propriedades CSS.
- `README.md` → Este documento explicativo.

---

## 📖 Objetivo
O objetivo é servir como um **catálogo prático** para aprender:
- Estrutura básica de documentos HTML.
- Tags semânticas, de texto, mídia, formulários, tabelas e interatividade.
- Propriedades CSS para tipografia, cores, box model, layout, animações e responsividade.

---

## 🧩 Conteúdo do HTML
O arquivo `index.html` demonstra:
- **Estrutura**: `<!DOCTYPE>`, `<html>`, `<head>`, `<body>`.
- **Semântica**: `<header>`, `<nav>`, `<main>`, `<article>`, `<section>`, `<aside>`, `<footer>`.
- **Texto**: `<p>`, `<h1>–<h6>`, `<strong>`, `<em>`, `<mark>`, `<blockquote>`, `<abbr>`, `<time>`, etc.
- **Mídia**: `<img>`, `<audio>`, `<video>`, `<canvas>`, `<svg>`, `<figure>`, `<iframe>`.
- **Formulários**: `<form>`, `<input>`, `<textarea>`, `<select>`, `<option>`, `<button>`, `<fieldset>`, `<legend>`, `<progress>`, `<meter>`.
- **Tabelas**: `<table>`, `<thead>`, `<tbody>`, `<tfoot>`, `<tr>`, `<th>`, `<td>`.
- **Interatividade**: `<details>`, `<summary>`, `<dialog>`.

---

## 🎨 Conteúdo do CSS
O arquivo `style.css` demonstra:
- **Seletores**: universais, por ID, classe, descendente, filho direto, pseudo-classes e pseudo-elementos.
- **Texto**: `font-size`, `line-height`, `letter-spacing`, `text-align`, `text-transform`, `text-decoration`.
- **Cores e fundo**: `background-color`, `background-image`, `background-size`, `opacity`.
- **Box Model**: `margin`, `padding`, `border`, `border-radius`, `box-shadow`.
- **Layout**: `display: flex`, `justify-content`, `align-items`, `display: grid`, `grid-template-columns`, `gap`, `position`, `float`, `clear`.
- **Transições e animações**: `transition`, `@keyframes`, `animation`.
- **Transformações**: `rotate`, `scale`, `translate`.
- **Responsividade**: `@media` queries.
- **Variáveis CSS**: `--cor-principal`, `var(--cor-principal)`.
- **Funções úteis**: `calc()`, `rgba()`, `linear-gradient()`, `radial-gradient()`.
- **Recursos modernos**: `filter`, `backdrop-filter`, `clip-path`.

---

## 🚀 Como Usar
1. Salve os arquivos `index.html` e `style.css` na mesma pasta.
2. Abra o `index.html` em um navegador.
3. Explore cada seção e veja como o CSS afeta os elementos.
4. Edite e experimente novas combinações para fixar o aprendizado.

---

## 📌 Observação
- Algumas tags antigas como `<center>`, `<font>`, `<big>` são **obsoletas** e não foram incluídas, mas podem ser estudadas separadamente.
- Este catálogo é para **aprendizado** e não deve ser usado como base de produção real.
